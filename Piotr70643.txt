Zadanie 10
Jeżeli mamy już przygotowane repozytorium i jesteśmy w odpowiednim folderze,
Komendą:
	git status
możemy sprawdzić czy w flderze mamy jakiś nowy lub modyfikowany plik.
Następnie pleceniem:
	git add (nazwa pliku)
informujemy który plik chcmy przesłać lub (.) kropką wybieramy wszystkie pliki
kolejnym krokiem jest zatwierdzenie pliku komendą
	git commit -m "(nasza wiadomość)"
i na sam koniec by załadować wszystkie pliki na serwer użwamy komendy:
	git push origin master
